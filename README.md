# FinanceTracker Mobile 💰

Aplicativo mobile de controle financeiro desenvolvido com Python, Flet e Matplotlib.

## Funcionalidades

✅ Adicionar gastos  
✅ Categorizar despesas  
✅ Salvar dados em JSON  
✅ Manter gastos após fechar o app  
✅ Visualizar gráfico de gastos por categoria  
✅ Dashboard com total gasto  

## Tecnologias

- Python
- Flet
- Matplotlib
- JSON
- Git
- GitHub
